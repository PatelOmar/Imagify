# Imagify
Image Repository Project

# Application is hosted at: 
  https://imagify-f5874.web.app/
  
# To run the app with React:
### Installation:
  **npm install**

###  To Start Server:
  **npm start**
    
### Create a firebase project:
  **Add Firebase credentials to src/firebase/firestore.js**
  
### To Visit App:
  **localhost:3000/**
    
# Design Notes:
## Technologies used:
  **React.js, Node js, Firebase**
## React.js
  For my front end I used React.js as it is one of the most powerful javascript libraries. I used it for it's resusable components which can updated whenever there is a state change.
