# Imagify
Image Repository Project

# Application is hosted at: 
  https://imagify-f5874.web.app/
  
# To run the app with React:
### Installation:
    **npm install**

###  To Start Server:
    **npm start**
    
### Create a firebase project:
    **Add Firebase credentials to src/firebase/firestore.js**
  
### To Visit App:
    **localhost:3000/**
